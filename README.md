# 🌹 Proyecto Antonella

Este es un detalle especial desarrollado como una **Progressive Web App (PWA)**. Al abrirse, despliega una animación de partículas tipo arena que forman un corazón con el nombre de **Kerly Antonella**.

## ✨ Características
- **Animación Matemática:** Utiliza ecuaciones paramétricas para dibujar el corazón.
- **Efecto Arena:** Caída de partículas con física de gravedad y opacidad.
- **Instalable:** Configurado como PWA para que pueda guardarse como una app en el celular.

## 📱 Cómo instalar en tu celular
Para que aparezca como una aplicación en tu pantalla de inicio:

### En Android (Chrome):
1. Abre el enlace del proyecto.
2. Toca los **tres puntos** arriba a la derecha.
3. Selecciona **"Instalar aplicación"** o **"Agregar a la pantalla de inicio"**.

### En iPhone (Safari):
1. Abre el enlace del proyecto.
2. Toca el botón de **Compartir** (cuadrado con flecha hacia arriba).
3. Busca y selecciona **"Agregar al inicio"**.

## 🛠️ Tecnologías usadas
- HTML5 / Canvas API
- JavaScript (Vanilla)
- CSS3
- Web App Manifest & Service Workers
